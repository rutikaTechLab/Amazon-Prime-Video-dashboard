# 📺 Amazon Prime Video Dashboard

An interactive **Amazon Prime Video Dashboard** built to analyze and visualize the Prime Video content library. This dashboard provides insights into movies, TV shows, genres, ratings, release years, countries, and other key metrics using interactive charts and filters.

---

## 📌 Project Overview

The Amazon Prime Video Dashboard helps users explore Prime Video's content collection through visual analytics. It provides a clear understanding of the platform's content distribution across different categories and supports data-driven insights.

---

## 🚀 Features

- 📊 Interactive Dashboard
- 🎬 Movies vs TV Shows Analysis
- 🌍 Content Distribution by Country
- 🎭 Genre-wise Analysis
- ⭐ Rating Distribution
- 📅 Release Year Trends
- 🔍 Search and Filter Options
- 📈 Easy-to-understand Visualizations
